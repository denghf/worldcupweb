嗨起来 · 管理后台响应式设计稿

7 个 admin 页面在手机 (390px) 和 Web (1280px+) 两种尺寸下的布局对照。手机端的核心改造：表格 → 卡片列表、双栏 → drill-down、弹窗 → 底部 sheet、宽按钮组 → 折叠菜单。导航采用「汉堡 + 全屏抽屉」。

01
表格卡片化
9 列排名表、玩家表等宽表格全部改为竖向卡片，关键信息前置，次要信息折叠。
02
双栏改 drill-down
下注详情、赛果查看等左右双栏改为列表 → 详情的两步导航，返回键回到列表。
03
弹窗底部化
编辑赔率、新建玩家、确认操作等弹窗在移动端改为从底部滑入的 sheet，单手可触达。
04
网格自适应
7/8 列赔率网格改为 3 列 + 折叠/横向 snap 滚动；3-4 列统计卡改 2 列。
